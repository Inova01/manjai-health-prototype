# ManjAI Website Prototype

This folder contains the ManjAI presentation website and software-simulation dashboard prototype.

## Open directly

Open `index.html` in a browser.

## Open with local server

Run:

```powershell
cd C:\Users\Asus\OneDrive\Thesis\manjai_site
..\.venv\Scripts\python.exe -m http.server 4173 --bind 127.0.0.1 --directory .
```

Then open:

```text
http://localhost:4173
```

## Dashboard login

Use the default demo fields already shown in the login form.

The dashboard is a software-simulation prototype for thesis presentation. It is not a clinical diagnostic tool.
